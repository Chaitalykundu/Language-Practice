| Operator | Operator's name | Example |
| -------- | --------------- | ------- |
| +        | Addition        | 5+7     |
| -        | Subtraction     | 7-5     |
| \*       | Multiplication  | 5\*7    |
| /        | Division        | 6/2     |
| \*\*     | Exponential     | 4\*\*2  |
| %        | Modulus         | 15%7    |
| //       | Floor division  | 15//7   |

floor division = quotient
modulus = remainder
